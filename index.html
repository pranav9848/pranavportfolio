About me

I am Pranav. I have 8+ years of experience as a Sr.Full stack developer working in enterprise software application development, testing, maintenance, and production support. Being a Full-Stack Java developer, I have contributed to entire application, let’s say frontend to backend. But my focus mainly lies in developing backend applications. I have used various backend technologies like java and spring boot for developing microservices along with monolithic applications using JSP Servlet. I have also interacted with database frameworks like hibernate, JDBC, JPA and Junit, Mockito for testing purposes. I have got the chance to work with both SQL and NoSQL databases. Coming to SQL databases I have worked with Postgres, MySQL, oracle and on NoSQL, I have worked with MongoDB, DynamoDB which is on AWS. I have written API for performing the CRUD operations using the Hibernate ORM Deployed Spring Boot-based microservices in AWS using JENKINS CICD pipelines. I also got exposure to DOCKER and KUBERNETES. I performed unit testing using Junit and integration testing using Mockito. On AWS, apart from Dynamo Db I have also worked with various services like IAM security group, EC2, S3 buckets, Cloud Watch, Route 53, and many services like this. I have used swagger for API documentation. Coming to Front end technologies, I have worked with React and Angular as well.

My job is to build your website so that it is functional and user-friendly but at the same time attractive. Moreover, I add personal touch to your product and make sure that is eye-catching and easy to use. My aim is to bring across your message and identity in the most creative way. I created web design for many famous brand companies.
